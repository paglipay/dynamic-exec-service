# Instructions for Saving Files Uploaded via Slack

This guide explains the step-by-step process for saving files uploaded via Slack into a specific local directory with a desired filename.

## Steps:

1. **Identify Uploaded File in Slack**
   - Locate the file in the Slack channel where it was uploaded.
   - Access the file metadata (file name, type, URL).

2. **Locate the File in Local Download Directory**
   - Slack files are downloaded automatically to a known directory, for example: `slack_downloads` within the project workspace.
   - Confirm the exact file name as saved.

3. **Create Destination Directory (If Not Exist)**
   - Check if the target directory (e.g., `Leichman CTC`) exists locally.
   - If it doesn't exist, create it.

4. **Move and Rename File**
   - Move the file from the `slack_downloads` folder to the target directory.
   - Rename the file as required during this move operation, e.g., `Main DMP Panel.jpg`.

5. **Verify the File Placement**
   - Confirm that the file exists in the new directory with the correct name.

---

Following these steps ensures that files uploaded via Slack are organized efficiently in local directories with appropriate names for easier reference and management.
