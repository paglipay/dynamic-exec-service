# Text File CRUD Plugin

## Module
`plugins.text_file_crud_plugin`

## Class
`TextFileCRUDPlugin`

## Allowed Methods
- `create_text`
- `read_text`
- `update_text`
- `delete_text`
- `list_text_files`

## Features
- Supports `.txt`, `.md`, `.json` extensions
- Supports nested relative paths within base_dir
- Blocks absolute paths and traversal outside base_dir

## Usage
Use for managing text-based files in allowed directories.

## Examples
- Creating a text file
- Reading a markdown document
