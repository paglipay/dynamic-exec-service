# Generated Math Plugin

## Module
`plugins.generated_math_plugin`

## Class
`GeneratedMathPlugin`

## Methods
- `multiply(a, b)`: Multiplies two numbers
- `greet(name)`: Generates a greeting message

## Usage
This plugin provides basic math operations and greeting functionality.

## Example
```json
{
  "module": "plugins.generated_math_plugin",
  "class": "GeneratedMathPlugin",
  "method": "multiply",
  "args": [4, 5]
}
```
